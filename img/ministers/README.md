# In-game "ministers" screenshots

These are shown on the KvK Title Application form (speedups.html) in the
**"Not sure which is which? See them in game"** accordion under the Titles
section. The form auto-picks the file that matches the reader's language.

Drop one screenshot per language here, named by its language code, as **.jpg**.
Use the full ministers screen with the two titles we care about highlighted
(top-left = Chief Minister, bottom-right = Noble Advisor).

| File | Language |
|------|----------|
| `en.jpg` | English |
| `ar.jpg` | Arabic |
| `zh.jpg` | Chinese (Simplified) |
| `zh-tw.jpg` | Chinese (Traditional) |
| `fr.jpg` | French |
| `de.jpg` | German |
| `id.jpg` | Indonesian |
| `it.jpg` | Italian |
| `ja.jpg` | Japanese |
| `ko.jpg` | Korean |
| `pl.jpg` | Polish |
| `pt.jpg` | Portuguese |
| `es.jpg` | Spanish |
| `th.jpg` | Thai |
| `tr.jpg` | Turkish |

If a language's file is missing it falls back to `en.jpg`; if that's also
missing, the accordion just stays hidden — no broken image. Filenames must
match exactly (lowercase).
